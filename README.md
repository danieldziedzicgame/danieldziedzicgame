## 🚪 Projekt 1: Escape House (Unity)

▶️ **[Kliknij tutaj, aby obejrzeć Gameplay na YouTube](https://www.youtube.com/watch?v=FUWT4VYUTXI&list=PLM1K2A0XObbM)**

**Created by:** Daniel Dziedzic  
**Email:** danieldziedzicgame@gmail.com  

*Escape House* is a fully original escape room game project, designed and developed entirely from scratch by myself. This project allowed me to experience the complete production pipeline—from coding gameplay mechanics to level design and 3D asset creation.  

**Key Features & Implemented Tasks:** * **Core Mechanics:** Programmed character movement logic and developed a fully functional inventory system.  
* **Interactive Environment:** Implemented advanced logic for interactive objects, including opening doors, searching hidden compartments, and core puzzle mechanics.  
* **3D Integration & Optimization:** Prepared 3D models for interaction, ensuring their full integration and technical optimization within the game engine.  
* **AI-Assisted Development:** Actively utilized Gemini and Claude to optimize code architecture, speed up prototyping, and ensure efficient debugging.

## 🏃‍♂️ Projekt 2: Kregle Dinozaury (Blender- zadanie na studia)

**Created by:** Daniel Dziedzic  
**Email:** danieldziedzicgame@gmail.com  
A university project created as an unconventional approach to a standard "bowling ball and pins" animation assignment. Rather than using typical objects, I brought my own unique vision to life. Instead of standard pins, the scene features custom, fully animated 3D dinosaur models. In place of a regular bowling ball, a meteorite crashes into the scene, scoring a spectacular "strike" on the herd of dino-pins.

**Technical Highlights:**
* **Custom 3D Modeling:** Designed and modeled the original dinosaur characters and the meteorite from scratch in Blender.
* **Character Animation:** Set up rigs and animated the dinosaurs (idle movements) to blend naturally with the environment.
* **Physics & Dynamics:** Configured Rigid Body physics to simulate a realistic, high-impact collision during the strike.
